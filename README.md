# Lives do IFPB
Um site que registra a lista com todas as lives do IFPB durante a quarentena

## Funcionalidades

- Sistema de login básico
- Painel administrativo
- Notificações push

## Instruções
### 1. Instalação
```bash
git clone https://github.com/josejefferson/ifpblives
cd ifpblives
npm install
```

### 2. Executar
```bash
npm start
```
